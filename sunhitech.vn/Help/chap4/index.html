IV.Quản lý Hỏi đáp

1. Trả lời câu hỏi

  • Vào D.Nội dung -> 4.Quản lý hỏi đáp

  • Chọn câu hỏi cần trả lời -> Trong cột Trạng thái nhấn trả lời để vào giao diện trả lời   -> Nhập nội dung câu trả lời -> Nhân nút lưu để hoàn tất

2.Phê duyệt câu hỏi

  • Vào D.Nội dung -> 4.Quản lý hỏi đáp
  • Chọn câu hỏi cần phê duyệt -> Nhấn nút phê duyệt

3.Xóa câu hỏi

  • Vào D.Nội dung -> 4.Quản lý hỏi đáp
  • Chọn câu hỏi cần xóa -> Nhấn nút xóa
